Introducción a la Ciencia de Datos

Introducción

La demanda de profesionales calificados en ciencias de datos en la industria, la academia y el gobierno está creciendo rápidamente. Este libro presenta conceptos y habilidades que pueden ayudarlo a enfrentar los desafíos del análisis de datos del mundo real. Cubre conceptos de probabilidad, inferencia estadística, regresión lineal y aprendizaje automático. También le ayuda a desarrollar habilidades como la programación R, la discusión de datos con dplyr, la visualización de datos con ggplot2, la creación de algoritmos con caret, la organización de archivos con UNIX/ Linux shell, el control de versiones con Git y GitHub y la preparación de documentos reproducibles con knitr y R markdown . El libro está dividido en seis partes: R, Visualización de datos, Disposición de datos, Estadísticas con R, __Aprendizaje automático__yHerramientas de productividad. Cada parte tiene varios capítulos destinados a ser presentados como una sola conferencia e incluye docenas de ejercicios distribuidos en capítulos.

Estudios de caso

A lo largo del libro, utilizamos casos de estudio motivadores. En cada estudio de caso, intentamos imitar de manera realista la experiencia de un científico de datos. Para cada uno de los conceptos cubiertos, comenzamos haciendo preguntas específicas y las respondemos mediante análisis de datos. Aprendemos los conceptos como un medio para responder las preguntas. Ejemplos de los casos de estudio incluidos en el libro son:

¿Quién encontrará útil este libro?

Este libro está destinado a ser un libro de texto para un primer curso de ciencia de datos. No es necesario tener conocimientos previos de R, aunque algo de experiencia en programación puede ser útil. Los conceptos estadísticos utilizados para responder las preguntas del estudio de caso se presentan solo brevemente, por lo que se recomienda encarecidamente un libro de texto de Probabilidad y Estadística para comprender en profundidad estos conceptos. Si lee y comprende todos los capítulos y completa todos los ejercicios, estará bien posicionado para realizar tareas básicas de análisis de datos y estará preparado para aprender los conceptos y habilidades más avanzados necesarios para convertirse en un experto.

¿Que cubre este libro?

Comenzamos repasando los conceptos básicos de R y el tidyverse. Aprendes R a lo largo del libro, pero en la primera parte revisamos los componentes necesarios para seguir aprendiendo.

La creciente disponibilidad de conjuntos de datos informativos y herramientas de software ha llevado a una mayor dependencia de visualizaciones de datos en muchos campos. En la segunda parte, demostramos cómo usar ggplot2 para generar gráficos y describir principios importantes de visualización de datos.

En la tercera parte demostramos la importancia de las estadísticas en el análisis de datos respondiendo preguntas de estudio de caso usando ** probabilidad, inferencia y regresión ** con R.

La cuarta parte utiliza varios ejemplos para familiarizar al lector con disputas de datos. Entre las habilidades específicas que aprendemos están el raspado web, el uso de expresiones regulares y la unión y remodelación de tablas de datos. Hacemos esto usando las herramientas tidyverse.

En la quinta parte presentamos varios desafíos que nos llevan a introducir aprendizaje automático. Aprendemos a usar el paquete caret para construir algoritmos de predicción que incluyen vecinos K más cercanos y bosques aleatorios.

En la parte final, proporcionamos una breve introducción a las herramientas de productividad que usamos diariamente en proyectos de ciencia de datos. Estos son RStudio, UNIX/ Linux shell, Git y GitHub, y knitr y R Markdown.

¿Qué no cubre este libro?

Este libro se centra en los aspectos de análisis de datos de la ciencia de datos. Por lo tanto, no cubrimos aspectos relacionados con la gestión de datos o la ingeniería. Aunque la programación R es una parte esencial del libro, no enseñamos temas informáticos más avanzados como estructuras de datos, optimización y teoría de algoritmos. Del mismo modo, no cubrimos temas como servicios web, gráficos interactivos, computación paralela y procesamiento de transmisión de datos. Los conceptos estadísticos se presentan principalmente como herramientas para resolver problemas y no se incluyen descripciones teóricas en profundidad en este libro.
